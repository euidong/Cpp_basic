# 변수와 메모리 주소

변수는 메모리의 길이, 메모리의 주소를 기억해야한다.<br>
여기서 메모리의 길이는 type으로 알 수 있다.


c++에서는 변수를 사용하기 전에 반드시 먼저 해당 변수를 저장하기 위한 메모리 공간을 할당받아야 한다.
`이것을 변수의 선언이라고 한다.`

# 기본 타입의 메모리 크기
short = 2byte
int = 4byte
long = 4byte
long long = 8byte
float = 4byte
double = 8byte
long double = 8byte
char = 1byte

auto = 자동으로 타입 선언.(배열로는 못써요.)
