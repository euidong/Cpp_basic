### 기초
```c++
class 클래스명
{
  public:
    //공개할 데이터 및 함수

  private:
    //공개하지 않고 클래스 내부에서만 사용할 데이터 및 함수
}
```

### 상속 형태
```c++
class 클래스명 : public 부모클래스명
{
  public:
    //공개할 데이터 및 함수

  private:
    //공개하지 않고 클래스 내부에서만 사용할 데이터 및 함수
}

//파생 클래스에서 부모 클래스의 멤버 함수를 사용하고 싶다면 부모클래스명::멤버함수 의 형태로 호출해야한다.
```
